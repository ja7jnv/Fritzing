** このリポジトリの使い方
* コミットの習慣
- .fzzの保存
  普通に保存すれば、このファイルが作られる。
- .fzの取り出し（または非圧縮保存）
  .fzzをフォルダ内にコピーし、サフィックスを.zipに変更する。
  その後、zipを解凍し、.fz ファイルを取り出す。
- 両方をGitでコミット
* .gitignore の設定
Fritzingが作成するバックアップファイル（末尾に ~ がつくものなど）がリポジトリに混入しないよう、.gitignore ファイルを作成して除外設定をしておく。
* 画像（PNG等）の同梱
回路図の画像も一緒にコミットしておく。GitHubのWeb画面上で「何を変えたか」がひと目で分かるようになる。
